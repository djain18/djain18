## Featured Projects

### [Rise: AI Internship Discovery Platform](https://github.com/djain18/internship-automation-pipeline) &nbsp; [![Live](https://img.shields.io/badge/live-rise--web-16a34a?style=flat-square)](https://rise-web-kappa.vercel.app)

Scrapes LinkedIn hiring posts every night, runs a four-provider LLM cascade to filter out scams and ineligible roles, and publishes clean, India-eligible listings to a site students can browse. A Python data pipeline and a FastAPI + React app that meet at a single Google Sheet, with 67 unit tests and an eval gate running in CI on every push.

`Python` ﾂｷ `FastAPI` ﾂｷ `React` ﾂｷ `Apify` ﾂｷ `LLM cascade (OpenRouter / Gemini / OpenAI / Groq)`

### [Lead Generation Pipeline](https://github.com/djain18/lead-generation-pipeline)

A B2B lead-sourcing engine that pulls prospects from Apollo.io, Google Maps (via Apify), JustDial, and IndiaMart, scores them on multiple factors, enriches phone numbers through website scraping and the Lusha API, and publishes a deduplicated, sorted CRM to Google Sheets.

`Python` ﾂｷ `Apollo.io` ﾂｷ `Apify` ﾂｷ `Lusha API` ﾂｷ `Google Sheets`
