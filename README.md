## What I've built

### [Rise](https://github.com/djain18/internship-automation-pipeline) · [live site](https://rise-web-kappa.vercel.app)
An internship-discovery platform that scrapes LinkedIn hiring posts every night, runs a four-provider LLM cascade to filter out scams and ineligible roles, and publishes clean, India-eligible listings to a site students can browse. A Python data pipeline and a FastAPI + React app that meet at a single Google Sheet, with 67 unit tests and an eval gate running in CI on every push.

### [Lead Generation Pipeline](https://github.com/djain18/lead-generation-pipeline)
A B2B lead-sourcing engine that pulls prospects from Apollo.io, Google Maps (via Apify), JustDial, and IndiaMart, scores them on multiple factors, enriches phone numbers through website scraping and the Lusha API, and publishes a deduplicated, sorted CRM to Google Sheets.
